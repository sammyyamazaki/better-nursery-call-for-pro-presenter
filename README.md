Better Nursery Call for ProPresenter

Features:
- Instant Number Display
- No need for Propresenter Operator to click anything
- Number only disappears when Kids Ministry deletes it
- Multiple Numbers can be shown at once
- Different Groups have different Prefixes
- Auto-Port Discovery: In case PP changes it's port the software will find the new port automatically
- Admin Panel with (Managed Groups, Delete all Calls, Settings)

How to (MacOS)
- Open Terminal
- Go the folder where the files are
- start server by typing: python3 server_v2.py
- open browser: localhost:8080/admin.html
- Set the Settings (IP, Port, Message Template) You may need to save Settings befor being able to select Template
- open second brower: localhost:8080
- done

Hope you enjoy
